# @nesoi/s3

[AWS S3](https://aws.amazon.com/s3/) plugin for the [nesoi](https://github.com/nesoi-org/nesoi) framwork, built with [aws-sdk-js](https://github.com/aws/aws-sdk-js).

## Install

```
npm i @nesoi/s3
```
